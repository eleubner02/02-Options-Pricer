# 02-Options-Pricer

## Objective: Build a Simple European option pricer using the Black-Scholes model

## Inputs: Stock Price, Strike Price, Time to Maturity, Risk-Free Rate, Volatility

## Outputs: The price of a Call option and the price of a Put option

## Key Functions to Build: A dunction for d1 and a function for d1, a function for the Call price, a function for the Put price
